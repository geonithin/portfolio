Technology Stack

Frontend Technologies

React Router v7 TypeScript Tailwind CSS Server-Side Rendering Vite

Built with the latest React Router v7 featuring built-in SSR capabilities, file-based routing, and type-safe routing. TypeScript provides 95%+ type coverage for enhanced code quality and IDE support. Tailwind CSS enables rapid UI development with a mobile-first responsive design approach.

Backend & Database

Supabase PostgreSQL Row Level Security Real-Time Subscriptions

Supabase provides a managed PostgreSQL database with instant RESTful APIs, real-time subscriptions for live updates, and built-in authentication extended with custom session management. Row Level Security ensures fine-grained data access control.

Database Schema (8 Tables):

  • • auth_users: User authentication and profiles
  • • books: Complete book catalog
  • • booking_requests: Book request workflow
  • • loan_records: Active and historical loans
  • • messages: Staff-student communication
  • • notifications: System-wide alerts
  • • reviews: User book reviews and ratings
  • • library_settings: System configuration

External Integrations

Google Calendar API Google OAuth 2.0 googleapis (v160.0.0)

Integrated Google Calendar API for automated due date reminders with OAuth 2.0 secure authentication. Features include token management with automatic refresh, secure storage, and graceful error handling.
