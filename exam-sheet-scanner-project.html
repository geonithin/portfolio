Project Overview

The Exam Sheet Scanner is a comprehensive system that leverages Google Gemini AI for OCR-powered data extraction from exam answer sheets. It supports multiple input methods including camera capture, file upload, and manual entry, with a hierarchical dashboard organized by Department, Year, and Section.

Key Features

Multi-Method Input

  • Camera capture for real-time scanning
  • File upload (image formats)
  • Manual data entry as fallback
  • Batch processing capabilities

AI-Powered OCR

  • Google Gemini AI (gemini-1.5-flash)
  • Intelligent text extraction
  • Automatic field recognition
  • High accuracy data parsing

Hierarchical Dashboard

  • Department → Year → Section view
  • Advanced search and filtering
  • Record management interface
  • Performance analytics

Data Management

  • PostgreSQL with JSONB storage
  • Flexible marks data structure
  • Auto-calculation of totals & grades
  • Indexed queries for fast performance

Data Structure

Exam Sheet Fields (13+ Fields)

  • Register Number — Student identification
  • Student Name — Full name
  • Department — Academic department
  • Year — Academic year
  • Section — Class section
  • Exam Type — Internal/External/etc.
  • Subject Code — Course identifier
  • Subject Name — Course name
  • Marks Data (JSONB) — Flexible storage
  • Total Marks — Sum of all marks
  • Grade — Final grade
  • Timestamps — Created/Updated dates
  • Image URL — Scanned sheet reference
