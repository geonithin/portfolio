# Exam Sheet Scanner Project Images

This directory contains images for the Exam Sheet Scanner project.

## Required Images

### Cover Image
- **cover.jpg** - Main project cover image for the portfolio
  - Recommended size: 800x600px
  - Should showcase the exam sheet scanning interface or dashboard

### Screenshots
Add screenshots showing:
- Dashboard view with hierarchical organization (Department → Year → Section)
- Camera scanning interface
- File upload interface
- Exam sheet data display
- Search and filter functionality
- AI OCR extraction in action

### Sample Exam Sheets
- Add sample scanned exam sheets (ensure no sensitive data)
- Show before/after OCR extraction

## Instructions
1. Add your project screenshots to this folder
2. Update the cover.jpg for the main project thumbnail
3. Reference additional screenshots in the project detail page if needed

## Image Naming Convention
- Use descriptive names: `dashboard-view.jpg`, `camera-scan.jpg`, `ocr-extraction.jpg`
- Keep filenames lowercase with hyphens
- Use .jpg for photos, .png for UI screenshots with transparency
